Documentation

With this API you can get all the information about your favorite movies!

Business logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies.
Return data about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, title, description, genres, director, image URL, whether it’s featured or not. Example:
{
  id : 1,
  title : "Oldboy",
  description : "...",
  genres: [
    {
        id: 1,
        name: "thriller",
        description: "..."
    }
  ],
  director: {
    id: 1,
    name: "Park Chan-wook",
    bio: "...",
    yearBirth: 1963,
    yearDeath: null
  },
  imageURL : "...",
  featured : false
}
Return data about a genre by genreName /movies/genres/[Genrename] GET None A JSON object holding data about the genres, containing ID, genreName and description. Example:
{
  id : 1,
  name : "thriller",
  description : "..."
}
Return data about a director by Name /movies/directors/[directorName] GET None A JSON object holding data about the director, containing ID, name, bio, birthyear and deathyear. Example:
{
  id : 1,
  name : "Wes Craven",
  bio : "...",
  yearBirth : 1939,
  yearDeath : 2015,
}
Add User /users POST A JSON object holding data about the user to add. Example:
{
  username : "Sam",
  email : "sam@gmail.com",
  password : "5g§j73kl%",
  birth_date : "1975-09-12",
  favoriteMovies : []
}
A JSON object holding data about the user that was added, including an ID:
{
  id : 5,
  username : "Sam",
  email : "sam@gmail.com",
  password : "5g§j73kl%",
  birth_date : "1975-09-12",
  favoriteMovies : []
}
Update the username of a user by user-id /users/[UserID] PUT None A JSON object holding the updated data about the user:
{
  id : 5,
  username : "Sam",
  email : "sam@gmail.com",
  password : "5g§j73kl%",
  birth_date : "1975-09-12",
  favoriteMovies : []
}
Allow users to add a movie to their list of favorites /users/[UserID]/[MovieTitle] POST None A text message indicating the movie was added to favoriteMovies of the user.
Allow users to remove a movie from their list of favorites /users/[UserID]/[MovieTitle] DELETE None A text message indicating that the movie was removed from favoriteMovies of the user.
Allow existing users to deregister /users/[UserID] DELETE None A text message indicating that the user was deleted.
