Documentation

With this API you can get all the information about your favourite movies!

Business logic URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies.
Return data about a single movie by title to the user /movies/[Title] GET None A JSON object holding data about a single movie, containing an ID, title, description, genre, director, image URL, whether itâ€™s featured or not. Example:
{
  id : 1,
  title : "Oldboy",
  description : "...",
  genre : "Thriller",
  director:
  {
      id: 1,
      name: "Park Chan-wook",
      bio: "...",
      yearBirth: 1963,
      yearDeath: null
  }
  imageURL : "...",
}
Return data about a genre by genrename /movies/[Genre]/[Genrename] GET None A JSON object holding data about the genre, containing ID, genreName and description. Example:
{
  id : 1,
  genreName : "Thriller",
  description : "...",
}
Return data about a director by Name /movies/[Directors]/[Name] GET None A JSON object holding data about the director, containing ID, name, bio, birthyear and deathyear. Example:
{
  id : 1,
  name : "Wes Craven",
  bio : "...",
  yearBirth : 1939,
  yearDeath : 2015,
}
Add User /users POST A JSON object holding data about the user to add. Example:
{
  name : "Sam",
}
A JSON object holding data about the user that was added, including an ID:
{
  id : 5,
  name : "Sam",
  favoriteMovies : []
}
Update the username of a user by username /users/[Username] PUT A JSON object holding data about the user to update. Example:
{
  name : "Sammmy",
}
A JSON object holding the updated data about the user:
{
  id : 5,
  name : "Sammmy",
  favoriteMovies : []
}
Allow users to add a movie to their list of favorites /users/[Name]/[Movies]/[MovieID] PUT A JSON object holding data about the movie to add. Example:
{
  title : "Taxi Driver",
}
A text message indicating the movie was added to favoriteMovies of th user.
Allow users to remove a movie from their list of favorites /users/[Name]/[Movies]/[MovieID] DELETE A JSON object holding data about the movie to remove and the user. Example:
{
  title : "Taxi Driver",
}
A text message indicating that the movie was removed from favoriteMovies of the user.
Allow existing users to deregister /users/[Username] DELETE A JSON object holding data about the user to remove. Example:
{
  user : "Sam",
}
A text message indicating that the user was deleted.
